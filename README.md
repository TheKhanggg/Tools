# 📥 TikTok Downloader Pro

**TikTok Downloader Pro** là một công cụ máy tính (Desktop App) mạnh mẽ, giao diện hiện đại được xây dựng dựa trên nền tảng **Python** và thư viện **CustomTkinter**. Phần mềm hỗ trợ tải video, âm thanh và slideshow ảnh từ TikTok với chất lượng cao nhất, hoàn toàn sạch watermark (logo/id) thông qua TikWM API tốc độ cao.

---

## ✨ Tính năng nổi bật

* **Tải Video Không Watermark:** Tải video TikTok gốc với độ phân giải cao nhất, không bị dính logo hay ID người dùng.
* **Hỗ trợ Bài đăng Ảnh (Slideshow):** * Tự động phát hiện và tải xuống toàn bộ chuỗi ảnh chất lượng cao.
    * Linh hoạt trích xuất riêng file âm thanh nền (MP3) từ các bài đăng slideshow.
* **Tách Âm Thanh (Audio Extractor):** Tải trực tiếp file nhạc nền (`.mp3` hoặc `.m4a`) từ bất kỳ video TikTok nào.
* **Chức năng Muted Video:** Loại bỏ hoàn toàn âm thanh của video gốc một cách nhanh chóng nhờ tích hợp lõi xử lý FFmpeg.
* **Giao diện Fluent Design Hiện Đại:**
    * Hỗ trợ đầy đủ chế độ Sáng/Tối (**Light/Dark/System Mode**) mượt mà.
    * Thanh tiến trình (**Progress Bar**) cập nhật dung lượng và phần trăm theo thời gian thực.
    * Xử lý đa luồng (**Threading**) giúp ứng dụng luôn mượt mà, không bao giờ bị đơ (Freeze) khi đang tải.
* **Tự động lưu cấu hình (`config.json`):** Ghi nhớ chế độ giao diện và thư mục lưu trữ yêu thích của bạn cho những lần khởi chạy sau.
