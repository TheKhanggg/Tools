## 🛠️ Hướng dẫn sử dụng & Cấu hình phụ trợ

### 1. Cách chạy ứng dụng
1. Tải file `TikTok_Downloader_Pro.exe` từ liên kết phía trên.
2. Di chuyển file vào một thư mục cố định trên máy tính của bạn.
3. Kích đúp chuột để mở và dán link TikTok vào để sử dụng.

### 2. Tính năng mở rộng (Tải Video Không Tiếng - Muted Video)
Để sử dụng chức năng nâng cao loại bỏ âm thanh khỏi video, ứng dụng yêu cầu máy tính của bạn cài đặt công cụ **FFmpeg**:
1. Tải FFmpeg bản dành cho Windows.
2. Thêm đường dẫn thư mục chứa file `ffmpeg.exe` vào biến môi trường hệ thống (**Environment Path**).
